After clicking, look for "Advanced" (Chrome), "Show Details" (Safari), or similar options, and then proceed to the server site.

Need Help?

• Make sure your Index Media Server is running

• Check that you're on the same network as the server

• Letters A-H, J-N, P-Z and numbers 0-9 (case insensitive)

• Contact your server administrator if you don't have a code
